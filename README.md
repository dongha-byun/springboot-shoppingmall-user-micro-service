# 사용자 / 인증 관련 마이크로 서비스

## 1. 개요
* 회원가입/로그인/이메일 및 비밀번호 찾기 등 일반 사용자의 계정과 관련된 업무를 수행하는 도메인 
* 로그인을 통한 Access Token 생성 및 Refresh Token 관리를 JWT 라이브러리를 통해 수행
